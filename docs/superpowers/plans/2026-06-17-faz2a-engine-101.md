## Global Constraints (101 ruleset — PO-approved defaults; all overridable via config)
- **Deck:** single 106-tile set (same as Klasik). **Deal:** 4 players, each **21**, starter (dealer's right) **22**; indicator flipped off the stock; thin draw stock remains. (Exact post-deal stock count verified at implementation: 106 − 1 indicator − (21·3 + 22) = 106 − 1 − 85 = **20** stock.)
- **No 13→1 wrap** (`runWrap13to1: false`).
- **Opening (el açma):** a player may lay melds only when their laid melds total **≥ 101** face value (okey/sahte-okey counts as the value it represents in that meld). Pairs route: open with **5 pairs** (`pairsOpenCount: 5`). Before opening, the hand stays concealed.
- **İşleme (lay-off):** after opening, add tiles to ANY table meld; cap **2 tiles per run per turn** (`layOffCapPerRun: 2`).
- **Açık taş / işlek penalty (PO rules — verbatim from spec §6):**
  - A non-çift player who takes the LEFT neighbour's discard MUST open immediately; if they open → that left neighbour is penalized **+101**; if they cannot open → the tile is returned.
  - A player who declared **"çifte gidiyorum"** may take the left discard and HOLD it; the moment they OPEN (≥5 pairs laid) → left neighbour **+101**.
  - **işlek tile discard** (discarding a tile an opponent immediately needs/uses) → **+101**; **discarding the okey mid-hand** → **+101**.
- **Scoring (penalty; lowest cumulative over 11 hands wins):**
  - Finisher: **−101** (credit). Opened-non-finisher: **+ sum of remaining tile face values**. Held okey/sahte-okey: **flat +101 each** (added to the sum). Never-opened: **+202**.
  - **Finish-type ladder (multiplicative on the finisher credit + non-finisher base):** ×2 for okey-finish / elden (going out in one turn) / çift; combinations ×4. Ladder values: −101 / −202 / −404 / −808.
  - **Riziko axis (orthogonal ×2):** if the indicator flipped is itself a false joker, the whole ladder doubles (−202 / −404 / −808).
  - **Flat penalties NOT multiplied:** işlek +101, wrong-okey-discard +101, floor-take-and-opened +101 are flat, each applies **once per hand** by default (`penaltyRepeatable: false`), summed.
  - **Çift declaration risk:** declared çift but NEVER opens → **2 × non-opener** (riziko-free 404, riziko 808). Opened ≥5 pairs but didn't finish → **2 × (remaining-tile sum)**.
  - Çift declaration is **binding** for the hand; a çift-declarer's floor-take is committed (no free lift-and-return; that courtesy is non-çift only).
- **Finish:** must retain a tile to discard (cannot finish with an empty rack).
- **Draw-pile exhaustion:** hand is scored normally (no −101 finisher credit; every non-finisher pays their applicable penalty). Not a void-replay (Klasik voids; 101 scores).
- Determinism; pure (no DOM/Date.now/Math.random); TDD; frequent commits; no placeholders.

### Task 2: `openingValue()` + open validation (el açma ≥101)

**Files:** Create `open.ts`; Test `open.test.ts`; export from index.
**Interfaces:**
- `function openingValue(melds: Tile[][], okey: Tile): number` — sum of face values of all tiles across the given melds; a wild (false joker or okey-valued tile) used in a meld counts as the value of the slot it fills (for a run, the missing number; for a group, the group's number). Returns the total.
- `function isValidMeldSet(melds: Tile[][], okey: Tile, config): boolean` — every meld is a valid run/group (≥3, no 13→1 wrap for 101).
- `function canOpen(melds: Tile[][], okey: Tile, config): boolean` — `isValidMeldSet && openingValue >= config.openingThreshold` (or the pairs route: exactly `pairsOpenCount` valid pairs).
