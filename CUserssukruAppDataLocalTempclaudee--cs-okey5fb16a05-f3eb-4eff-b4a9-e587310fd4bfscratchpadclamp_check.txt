CLAMP FORMULA ANALYSIS
======================

Desktop (1280px+):
- vw is about 12.8px (1280/100)
- --tile-w: clamp(20px, 3.28vw, 42px)
  = clamp(20px, 3.28 * 12.8px, 42px)
  = clamp(20px, 41.984px, 42px)
  = 41.984px ✓ Correct (≈42px as intended)

- --tile-h: clamp(28px, 4.53vw, 58px)
  = clamp(28px, 4.53 * 12.8px, 58px)
  = clamp(28px, 57.984px, 58px)
  = 57.984px ✓ Correct (≈58px as intended)

- --tile-font: clamp(13px, 1.72vw, 22px)
  = clamp(13px, 1.72 * 12.8px, 22px)
  = clamp(13px, 22.016px, 22px)
  = 22px ✓ Correct

  = 20px ✓ Gracefully shrinks to minimum

- --tile-h: clamp(28px, 4.53vw, 58px)
  = clamp(28px, 4.53 * 4.8px, 58px)
  = clamp(28px, 21.744px, 58px)
  = 28px ✓ Gracefully shrinks to minimum

  = clamp(13px, 1.72 * 4.8px, 22px)
